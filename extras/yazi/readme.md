# Conifer (Yazi)

## Installation

1. Download the theme file to your Yazi `flavors` folder (e.g., to
   `~/.config/yazi/flavors/conifer.yazi/`; you may need to create the
   directory if you haven't already).
2. In your Yazi theme file (e.g. `~/.config/yazi/theme.toml`),
   add the following line:

```toml
[flavor]
dark = "conifer"
```
